buddyEngine

A C++ based, 2D game engine.

Acknowledgements

  • Uses SFML (Simple Fast Multimedia Library) by Laurent Gomilla et al.
  • Uses tinyXML-2 by Lee Thomason.

Install

Visual Studio 16 2019 + CMake GUI

  1. Clone or download buddyEngine
  2. Download the latest release of SFML to the same location as buddyEngine.
  3. Use the CMake GUI to configure SFML as per their tutorial but with additional steps:
    Set BUILD_SHARED_LIBS to FALSE
    Set SFML_USE_STATIC_STD_LIBS TO TRUE
  4. Open SFML.sln and build for both Debug and Release
  5. Build front.sln and run/debug.
